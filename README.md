# Northwind Data Warehouse Project

Welcome to the **Northwind Data Warehouse Project** repository! 

This project demonstrates an end-to-end data warehouse built with SQL Server, 
following the Medallion Architecture (Bronze → Silver → Gold).

---

## Objective
Develop a modern data warehouse using SQL Server to consolidate Northwind 
Traders' operational data, enabling clean and structured data for business use.

---

## Architecture (Data Engineering)
- **Bronze Layer**: Raw data loaded directly from the Northwind source database.
- **Silver Layer**: Cleaned and standardized data.
- **Gold Layer**: Business-ready data modeled into a star schema.
